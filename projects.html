JSON Data Example

Example of a JSON data structure for your project, with inline instructions:

{
  "jsonData": [
    {
      // Your title, this is an example
      "tittle": "TITLE | DISTRITO | INCMV AWARDS 2025",
      "subTittle": "Facebook",
      // [name image format] must be jpg
      "image": "22.jpg",
      // Your link, for example YouTube
      "link": "https://youtu.be/OwZJkrPtMjM?si=qLlSW8KVk03MbkQz&t=1",
      "allowEdit": false,
      "redirectPage": true,
      "page": "watch",
      "redirect_text": "Sending View From Youtube",
      "different_link_per_data": false,
      "different_img_per_data": false,
      // Your datatype, can be 'firebase' or 'azure.app' or others
      "viewDataType": "firebase",
      // You can change this if you use Azure or another backend
      // "viewDataType": "azure.app",
      "viewData": [
        // Your view data in Firebase Config Example
        // 'lokal' is your local or district, change it to match your firebase config
        // 'duration' is how much duration you want
        // 'image' your image name, must be jpg
        // 'views' you can change this if you are not using a database
        {"lokal": "Aguilar", "duration": "4:45", "link": "", "image": "22.jpg", "views": 0}
      ]
    }
  ]
}

OpenAI Token Key Example (Python) for your Microsoft Azure OpenAI

If you’re brave enough to take on AI, I suggest trying Microsoft OpenAI.

import json
import os
from openai import OpenAI

# ====== SETUP ======
# Your OpenAI API key (token)
os.environ["OPENAI_API_KEY"] = "your-openai-api-key-here"

# Path to your views.json
file_path = "views.json"

# ====== LOAD JSON ======
with open(file_path, "r") as f:
    views_data = json.load(f)

# ====== AI UPDATE INSTRUCTION ======
instruction = "Increase all 'view_count' fields by 1 and keep everything else unchanged."

# ====== OPENAI CALL ======
client = OpenAI(api_key=os.environ["OPENAI_API_KEY"])

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {"role": "system", "content": "You are a JSON editor. Only output valid JSON."},
        {"role": "user", "content": f"Here is the JSON:\n{json.dumps(views_data, indent=2)}\n\n{instruction}"}
    ],
    temperature=0
)

# ====== PARSE AI OUTPUT ======
updated_json_str = response.choices[0].message.content
updated_views = json.loads(updated_json_str)

# ====== SAVE BACK TO FILE ======
with open(file_path, "w") as f:
    json.dump(updated_views, f, indent=2)

print("✅ views.json updated successfully!")
