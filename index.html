Example Token JSON

Here is an example of a token JSON structure that you can use in your projects when using AI count:

{
        "bos_token": {
            "content": "<|startoftext|>",
            "lstrip": false,
            "normalized": false,
            "rstrip": false,
            "single_word": false
        },
        "eos_token": {
            "content": "<|return|>",
            "lstrip": false,
            "normalized": false,
            "rstrip": false,
            "single_word": false
        },
        "pad_token": {
            "content": "<|endoftext|>",
            "lstrip": false,
            "normalized": false,
            "rstrip": false,
            "single_word": false
        }
    }
